## Alumno.h
La clase Alumno se utiliza para representar la información de un estudiante, incluyendo su identificador, nombre, edad y estatura. Los constructores permiten crear objetos Alumno y establecer sus atributos al momento de la creación. Puedes utilizar esta clase para crear objetos que almacenen información sobre alumnos en un programa.

## mainMonticulo.cpp
El código proporcionado se centra en la manipulación de un heap (montículo) y muestra cómo se pueden utilizar las funciones make_heap, push_heap, pop_heap y sort_heap de la biblioteca estándar de C++ para trabajar con un heap (estructura de datos tipo árbol).

## mainColadePrioridad1.cpp
Demuestra cómo utilizar una cola de prioridad para administrar elementos con prioridad en orden descendente. La cola de prioridad garantiza que el elemento con la mayor prioridad siempre esté en la parte superior. El código muestra cómo agregar elementos, acceder al elemento con la mayor prioridad y eliminar elementos de la cola de prioridad.

## mainColadePrioridad2.cpp
utiliza colas de prioridad con funciones de comparación personalizadas para administrar una colección de objetos de la clase "Alumno". Se incluyen tres clases de comparación personalizadas (comparaEstatura, comparaNombre y comparaEdads) que permiten priorizar elementos de la clase "Alumno" según diferentes criterios: estatura, nombre y edad. El programa crea seis objetos "Alumno" con diferentes atributos y demuestra cómo utilizar colas de prioridad para ordenarlos según estos criterios. El usuario puede descomentar la sección relevante del código para probar la cola de prioridad basada en el criterio de su elección.
