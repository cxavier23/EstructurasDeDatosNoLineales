# Algoritmo de inserción y búsqueda en un árbol B
### //author: Christopher Xavier Sanchez Duran
### //UAMC

## main.cpp
Insertan elementos en un árbol B, se imprime el árbol en orden creciente y se realizan búsquedas en el árbol para encontrar elementos específicos. También maneja duplicados en la inserción.

## ArbolB.h
Este código implementa las operaciones de inserción y búsqueda en un árbol B, y también permite imprimir las claves en orden creciente. El árbol B es una estructura de datos utilizada para organizar y buscar datos de manera eficiente.

## Pagina.h
Esta clase representa un nodo de un árbol B y proporciona métodos y atributos para gestionar las claves y los punteros a los nodos hijos. Los árboles B se utilizan comúnmente en estructuras de datos para mantener y gestionar datos ordenados.
