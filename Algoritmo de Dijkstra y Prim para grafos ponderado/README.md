# Algoritmo de Dijkstra y Prim para resolver problemas relacionados con grafos ponderado
### //author: Christopher Xavier Sanchez Duran
### //UAMC

## main.cpp
El programa carga un grafo desde un archivo y calcular un árbol de expansión mínima utilizando el algoritmo de Prim. 

## MatAdyacencia
La clase proporciona una representación de grafo con muchas operaciones comunes, como agregar vértices y aristas, realizar recorridos y calcular caminos más cortos. Además, ofrece implementaciones de los algoritmos de Dijkstra y Prim para resolver problemas relacionados con grafos ponderados.
