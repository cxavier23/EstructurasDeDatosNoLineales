## main.cpp
Trabajar con árboles binarios de búsqueda (ABB) para elementos de tipo carácter y enteros. Realiza inserciones, búsquedas y eliminaciones de elementos en los árboles ABB y muestra el contenido de los árboles después de cada operación. Este código es un ejemplo práctico de cómo se pueden utilizar los ABB en C++ para gestionar datos ordenados de manera eficiente.

## ABB.h 
La clase de Árbol Binario de Búsqueda implementa la inserción, búsqueda y eliminación de elementos en un árbol binario de búsqueda de manera genérica. Puede funcionar con diferentes tipos de datos gracias al uso de plantillas. Este tipo de estructura de datos es útil para mantener datos ordenados y realizar búsquedas eficientes en ellos.
