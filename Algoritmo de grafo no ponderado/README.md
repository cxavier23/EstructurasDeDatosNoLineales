## main.cpp
Crea un grafo ponderado representado por una matriz de adyacencia y agrega algunos vértices y aristas. Luego, imprime la matriz de adyacencia resultante. Este tipo de estructura se utiliza comúnmente para representar relaciones entre elementos en una variedad de aplicaciones, como redes de transporte, redes sociales, sistemas de información geográfica

## main2.cpp
El código crea un grafo no ponderado representado por una lista de adyacencia y agrega algunos vértices y aristas. Luego, imprime la lista de adyacencia resultante. Las listas de adyacencia son una estructura de datos comúnmente utilizada para representar grafos, especialmente en el contexto de algoritmos de búsqueda y recorrido de grafos.

## MatAdyacencia.h
Iimplementación simple y flexible de una matriz de adyacencia para representar un grafo. Puede ser utilizado para grafos dirigidos o no dirigidos, ponderados o no ponderados. Los tipos de datos utilizados para los vértices y los pesos son parámetros de plantilla, lo que permite la flexibilidad en el tipo de datos que se utiliza para construir el grafo.

## ListaAdyacencia
Este código es flexible y puede ser utilizado para representar grafos dirigidos con cualquier tipo de datos, siempre que los tipos de datos utilizados admitan la igualdad y la impresión (lo que generalmente es el caso para tipos como int, char y string). Las funciones proporcionadas permiten crear, conectar y visualizar los vértices y las aristas del grafo.
