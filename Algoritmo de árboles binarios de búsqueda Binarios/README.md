## main.cpp
Crea un árbol B y realiza algunas operaciones en él, como la inserción de valores y la búsqueda de valores específicos. Luego, muestra el contenido del árbol en orden creciente y muestra el contenido de una página específica en la que se encuentra el valor 8. También intenta buscar un valor que no está presente en el árbol('100').

## NodoBin.h
La clase sirve para crear nodos de un árbol binario con elementos de tipo T, y proporciona métodos para establecer y obtener los elementos y punteros izquierdo y derecho de los nodos. Este código es una parte esencial de la implementación de estructuras de datos basadas en árboles binarios.

## AB.h
la clase AB permite la creación y manipulación de árboles binarios de búsqueda. Los métodos de inserción y recorrido se implementan de manera recursiva para proporcionar una funcionalidad completa para trabajar con árboles binarios de búsqueda. Este código se puede utilizar como base para construir y manipular árboles binarios de búsqueda.
