## main
Utiliza una clase llamada MatAdyacencia para trabajar con un grafo y realizar operaciones como cargar un grafo desde un archivo, imprimir su matriz de adyacencia y realizar el algoritmo de Dijkstra en ese grafo.

El programa parece recibir un argumento desde la línea de comandos que se espera sea el nombre de un archivo. El grafo se carga desde ese archivo y luego se imprimen su matriz de adyacencia y se ejecuta el algoritmo de Dijkstra desde el vértice 3 hacia un destino (vertice 6969).
